1) Building binary rpm package.
Execute the below command with root privileges.
Notice that we do not support binaries generated by this method.

  # rpmbuild --rebuild --define "kversion <target kernel version>" -ba --target <target arch>  <source rpm file>

ex) If you build a binary rpm using a source rpm hfcldd-4.8.22.4354-1.el8.src.rpm, execute this command.
  # rpmbuild --rebuild --define "kversion `uname -r`" -ba --target x86_64 hfcldd-4.8.22.4354-1.el8.src.rpm

2) You can install the rpm package generated in /root/rpmbuild/RPMS/<arch> directory.

ex)
# ls /root/rpmbuild/RPMS/x86_64/
hfcldd-4.8.22.4354-1.el8.x86_64.rpm
hfcldd-debuginfo-4.8.22.4354-1.el8.x86_64.rpm
hfcldd-debugsource-4.8.22.4354-1.el8.x86_64.rpm
kmod-hfcldd-4.8.22.4354-1.el8.x86_64.rpm
kmod-hfcldd-debuginfo-4.8.22.4354-1.el8.x86_64.rpm
